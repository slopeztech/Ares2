## Notes on CERT-14.3 (`static` local arrays)

All array declarations promoted to `static` belong to functions that are:

- **AMS engine methods** (`*Locked` suffix) ‚Äî always called while `mutex_` is held; effectively single-threaded.
- **API handler methods** ‚Äî the REST server processes one request at a time (documented in `api_server.cpp` line 54‚Äì57: *"The API server is single-threaded (one request at a time)"*).

In both contexts there is no re-entrancy, so function-local `static` storage is safe.

## Notes on CERT-16.1 (float comparison)

`std::numeric_limits<float>::min()` equals `FLT_MIN` ‚âà 1.175 √ó 10‚Åª¬≥‚Å∏ (minimum
positive normalized `float`).  The new guard rejects zero and subnormal values as
divisors ‚Äî a stricter and safer criterion than exact zero equality.
