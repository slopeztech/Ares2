# ARES v2.4.1 — Changelog

**Released:** 2026-06-02
**Previous version:** 2.4.0
**Type:** Patch — A2-4 null-iface driver detection at activation time.

---

## Summary

This patch adds **A2-4: null-iface early rejection**.  `activate()` now
validates that every driver referenced by `include … as ALIAS` statements
has a live interface pointer (`iface != nullptr`) before the engine reaches
`LOADED` state.

Previously, a driver whose `iface` had not been patched (e.g. a sensor whose
hardware initialisation failed) would be silently skipped at runtime: HK
fields would simply not appear in telemetry with no diagnostic.  This patch
turns that silent degradation into a hard activation failure with a
descriptive error message, so the problem is visible immediately.

Key highlights:

- **`validateAliasIfacesLocked()`** — new private method called from
  `activate()` after a successful parse; checks every alias's driver `iface`.
- **Descriptive error** — sets `lastError` to
  `"driver for alias 'X' has no interface (iface=null)"` and returns the
  engine to `IDLE`.
- **1 new lifecycle test** — `test_activate_null_baro_iface_rejected`
  confirms `activate()` returns `false` and status stays `IDLE`.

After this release the master CI pipeline continues to report
**Pass = 7 / Warn = 0 / Fail = 0 (OVERALL PASS)**.

## Changes

### 1. `validateAliasIfacesLocked()` (`src/ams/mission_script_engine.cpp`)

New private method added just before `activate()`:

```cpp
bool MissionScriptEngine::validateAliasIfacesLocked()
{
    for (uint8_t i = 0U; i < program_.aliasCount; i++)
    {
        const AliasEntry& ae = program_.aliases[i];
        bool ifaceNull = false;

        switch (ae.kind)
        {
            case PeripheralKind::GPS:
                ifaceNull = (ae.driverIdx < gpsCount_)
                            && (gpsDrivers_[ae.driverIdx].iface == nullptr);
                break;
            case PeripheralKind::BARO:
                ifaceNull = (ae.driverIdx < baroCount_)
                            && (baroDrivers_[ae.driverIdx].iface == nullptr);
                break;
            case PeripheralKind::COM:
                ifaceNull = (ae.driverIdx < comCount_)
                            && (comDrivers_[ae.driverIdx].iface == nullptr);
                break;
            case PeripheralKind::IMU:
                ifaceNull = (ae.driverIdx < imuCount_)
                            && (imuDrivers_[ae.driverIdx].iface == nullptr);
                break;
            default:
                break;
        }

        if (ifaceNull)
        {
            char msg[64];
            (void)snprintf(msg, sizeof(msg),
                "driver for alias '%s' has no interface (iface=null)", ae.alias);
            setErrorLocked(msg);
            return false;
        }
    }
    return true;
}
```

Called from `activate()` immediately after `loadFromStorageLocked()` succeeds:

```cpp
if (!validateAliasIfacesLocked())
{
    deactivateLocked();
    return false;
}
```
